# Aplicacion Web PackFlow

PackFlow es una aplicacion web orientada a servir de apoyo a microempresas y pequeñas empresas dedicadas al rubro de la venta y distribucion de empaques (cajas y bolsas de diferentes medidades y materiales de fabricacion) personalizables usando serigrafia e impresion offset. La aplicacion le dara la capacidad al usuario de poder consultar, disminuir e incrementar las cantidades de stock que posee en su negocio.
